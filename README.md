# Enhanced Transformer-Based Sentiment Analysis using Multi-Task Learning

## Overview
This project presents a transformer-based sentiment analysis framework that performs sentiment classification and sarcasm detection using multi-task learning. The model is trained on Amazon product reviews and supports aspect-based sentiment analysis.

## Features
- Sentiment Classification
- Sarcasm Detection
- Aspect-Based Sentiment Analysis
- Transformer-Based Architecture
- Multi-Task Learning

## Technologies Used
- Python
- PyTorch
- Transformers
- Pandas
- NumPy
- Scikit-learn

## Dataset
- Amazon Product Reviews Dataset
- Approximately 50,000 reviews
